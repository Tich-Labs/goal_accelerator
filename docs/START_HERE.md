# 🎯 Start Here

Welcome to Goal Accelerator PWA! This guide will get you up and running in 35 minutes.

## What Is This?
A minimalist, mobile-first PWA that helps you:
- Define yearly goals (max 6)
- Build daily habits
- Track progress
- Reflect weekly, monthly, yearly
- Stay focused and prevent overwhelm

**Built with:** HTML + Vanilla JS + Tailwind CSS + Supabase
**Cost:** $0/month (free tier)
**Status:** ✅ Production ready

---

## 🚀 Setup in 3 Phases

### Phase 1: Connect Supabase (20 min)
[SUPABASE_SETUP.md](./SUPABASE_SETUP.md)
- Create Supabase project
- Add credentials to app
- Create database tables
- Configure authentication

### Phase 2: Test Locally (10 min)
```bash
npm run serve
# Visit http://localhost:8080
# Create test account
```

### Phase 3: Deploy (5 min)
[GITHUB_PAGES_DEPLOYMENT.md](./GITHUB_PAGES_DEPLOYMENT.md)
- Push to GitHub
- Enable GitHub Pages
- Configure redirect URLs
- Done! 🎉
